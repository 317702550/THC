Procesador: El “cerebro” de la computadora. Todos los procesos de un aparato 
electrónico son llevados a cabo por el procesador. 

Kernel: El núcleo del sistema operativo. Tiene control sobre todo lo que se 
lleva a cabo dentro de la computadora. 

Linux: Sistema operativo de software libre más usado. 

Python: Lenguaje de programación de alto nivel utilizado por su fácil 
comprensión. 

Comando: Instrucciones u órdenes que el usuario manda a través de un shell 

Sistema operativo: También llamado software, es el programa que proporciona 
al usuario el acceso interactivo con los demás programas del dispositivo 

Repositorio: Base de datos con acceso a archivos digitales 

Servidor: Software que realiza tareas en nombre del usuario utilizándolo 

Cliente: Dispositivo o aplicación el cual está en comunicación con el servidor 

Ruta absoluta: La dirección completa desde el home del usuario hasta el archivo

Ruta relativa: La dirección a partir de un directorio o repositorio en común de dos o más ordenadores

Función: Bloque de código que recibe y procesa información

Lista = Objeto en Python que almacena en conjunto otros objetos. Modificable

Tupla = Igual a las listas, pero no se pueden modificar

Cadena = Objeto en Python dado por caracteres entre comillas

GitHub: Plataforma en línea  en la que se almacenan proyectos y el desarrollo de los
mismos.

Cadena multilínea: Conjunto delimitado por varias comillas

Terminal: Intérprete de comandos

Objeto: Unidad de información con la capacidad de ser modificada o alterada
mediante métodos
